# Panel_y `proto_3_1b`
## アプリ紹介

- 対象: 波形ビューア（Dash + Plotly + Parquet）
- 目的: 大容量データでも操作性を落とさず可視化・解析する

---

# 概要

`proto_3_1b` は、`proto_3_1` の機能を維持しつつ  
**大容量データ対応を強化した波形ビューア**です。

- 大容量ログ（目安: 数十万〜数百万点）でのハング対策
- 波形表示・計測・FFT・作図設定保存/復元を1画面で実行
- 解析系は生データ参照、描画系は最適化描画で両立

---

# 技術ハイライト

- `minmax_envelope` のベクトル化で描画前処理を高速化
- `np.searchsorted` で表示区間のインデックス抽出を軽量化
- 読み込み時 `float32` 変換でメモリ使用量を抑制
- 全体表示用 `df_overview` キャッシュで初期表示を高速化

---

# 各機能 1/5: 波形表示・行レイアウト

- 行ごとにチャンネルを自由に割り当て（1行複数ch重ね表示）
- 行追加/削除、Y軸範囲（min/max）指定
- `line` / `step` 表示切替（チャンネル単位）
- 色・線幅のチャンネル別カスタマイズ

---

# 各機能 2/5: 操作同期・表示制御

- ホバー位置を全行同期（スパイクライン表示）
- ズーム/パンのX軸範囲を全行同期
- スケール固定トグル（スクロールズームON/OFF）
- ライト/ダークテーマ切替
- ヘッダーピン固定、更新ショートカット（`Ctrl+Enter`）

---

# 各機能 3/5: 計測モード（Cursor A/B）

- 計測モードON時、クリックでカーソル A/B を交互配置
- `Δt`（時間差）と周波数換算を表示
- 区間統計をチャンネル横断で表示
  - A値/B値/Δ
  - Mean / Max / Min / P-P / RMS / Slope

---

# 各機能 4/5: FFT解析

- 2カーソル間を対象にFFT実行
- 対象チャンネルは最大3chまで選択
- 窓関数: Hanning / Hamming / Rectangular
- 表示: Amplitude / dB、周波数レンジ指定（min/max）

---

# 各機能 5/5: 作図設定の保存・ロード

- `.pyc.json` で作図状態を保存/復元
- 保存対象:
  - 行構成（channels, step, ymin/ymax）
  - 色・線幅
  - テーマ
  - スクロールズーム状態
- ロード時はデータ読込 + UI + 波形を一括復元

---

# 今後の機能追加予定（Roadmap）

- 次フェーズは「多カーソル化」「信号処理の高度化」「操作自動化」
